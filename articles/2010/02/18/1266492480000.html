2009 年个人回忆与总结


     out 了。看了电视剧《蜗居》后，从绝望中似乎看到了一丝希望。但总是在猜疑，总是在不确定性下执行着人生的算法。

转型期？

     毕业了，失业了。毕业后辞去了 Jinfonet 可提供的任职，“毅然”的年轻人敢闯。从 08 年到现在，转眼间已经从 Hand 到了 Jinfonet，再到 xxx，再到....。从 8 月份至 11 月，每月都投一些简历到昆明各公司。到云电信通 项目部试用两个星期后转入 研发部，两个月后辞了。适应不了的事太多，下面是在云电信通时提交的工作记录：

  • 信通试用（2009-8-31 至 2009-9-6）
  • 信通试用（2009-9-7 至 2009-9-13）
  • 信通试用（2009-9-14 至 2009-9-20）
  • 信通试用（2009-9-21 至 2009-9-27）

     辞职一天信通技术负责人、研发部经理、HR 谈了很多，也学习到了很多，非常感谢他们对我这两个月的照顾。
     混乱中、挫折中、失败中总感叹自己的渺小，耳边总想起那个声音：“那是必须的！ ”。
     自己究竟在追求什么，自己现在到底拥有什么，自己将何去，何从？耳边还是那个声音：“小虾米，……”。

纠结

     闲了，压力之余，看看影视剧。以往的影片，总是给人思考的余地。《蜗居》将赤裸裸的现实摆在了我的眼前。突然想起了《当幸福来敲门》，果真《蜗》是具有中 国特色的现实版。是应该好好思考一下自己的现实处境了。
     生老病死是自然规律，命。送走的时候固然一泻而出：情。
     情？要是能论述清楚，世界太简单了，生活也将变得没有意义。
     顾世点，隐忍。忍了自己的理想、忍了自己垂手可得的“秋收”。理想太遥远，那“秋收”呢？也不去收割？不过，我还没有秋收。

希望？

     在失业近两个月后的 11 月末，在昆明找到了一项工作，海归创业。海归创业，还是难 。公司目前四个股东，两个澳洲回来的海归，两个本地 投资。我，还是虾兵蟹将一个。某次和两个海归的同辈人谈了一下。难，呵呵。想玩，玩到大。但想在我拼命的时候能隐约看到每个人都拼命的情景，不为了私，为 的是中国软件。
     这段工作时间，到客户那边谈了两个项目，其中一个 去了大理。我，应该、必须时刻记得自己的责任。

     在不断自我否定中开展项目是不明智的，项目成败，取决于核心人员的性格。我再这么“飘”，项目失败是迟早的事。所以，负责地做出决定是我在接下来的日子里 必须做到的。唉，还是不成熟。

技术

     2009 年，感觉从云计算 中的飘渺找到了技术基石——NoSQL 。突然间，似乎感觉自己了解了程序的本质，Key-Value。
     从我所了解的程序语言看，都是变量-值所组成的。任何程序设计、实现都是基于 Key-Value 的，只是抽象程度不同而已。很怀念函数返回码的形式　:-)
     在逐步放弃 Java EE（CDI 、JSF 2） 的同时，回归了 Web 程序的本质—— HTML 。HTML + CSS  + JavaScript 才是王道。希望 HTML 5 赶快到来，让浏览器客户端与服务器的交互变得更易用 。

     一直以来对 JSP 充满了厌恶，对纯净的 Web（HTML + CSS + JavaScript） 充满了向往。其实，只要菜单舞得快，武林高手也是很难近身的。

社会活动？

     很久没有非常活跃在开源第一线 了……不过，2009 年也参与了一些活动：

  • 2009 中国软件技术英雄会（CSDN）
  • DevWOW 微软博客达人优秀奖
  • 最受欢迎的中国 50 技术博客（第 26 名，ZDNet）
  • CSDN 特约记者

     虽然没有参与第一线的开源活动，但目前的工作环境是纯开源的：Ubuntu + NetBeans + SVN + OOo。参与了所在公司开发环境从无到有的过程。

以后

     鲁迅说：“人生的旅途，前途很远，也很暗。然而不要怕，不怕的人的面前才有路。” 对，机会不是碰到的，是自己创造的，人定胜天。
