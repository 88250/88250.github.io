# B3log Announcement # B3log # GAE # Open Source # Life in Programming

----

Update: Oct 24, 2013

b3log 二级域名申请已经关闭，谢谢理解。

----

 

 

亲，还在为别人不能不能访问你的博客而苦恼？

亲，你不绑定自己的域名？

那，亲，不要再犹豫了，快发邮件给我申请 b3log.org 二级域名吧！

 

----

邮箱：DL88250@gmail.com

邮件标题：申请 b3log.org 二级域名

内容提要：

  • 你的应用 id
  • 需要的二级域名

发完邮件后记得把应用共享给我。

--

应用共享方法：

1. 点此登录 GAE 管理控制台，进入要申请二级域名的应用

2. 左侧点击 Administration -> Permissions 一栏

3. 发邀请给我吧，Owner 权限哦 ;-)

----

 

P.S. 亲，绑定新域名后记得：

  1. 修改 latke.props 配置，重新上传应用
  2. 修改偏好设定中的博客地址
